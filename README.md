# XlsAndLocalizable
andrid、iOS国际化字符串与xlsx互转


pip install xlrd
pip install xlsxwriter

-r 资源目录
-f 文件地址
-s 保存目录
-t 类型（iOS或android） 

python ToXlsx.py -r 资源文件夹地址 -s 输出地址 -t 类型
python ToXlsx.py -r 资源文件夹地址 -s 输出地址 -t 类型
python ToLocalizable.py -f xlsx地址 -s 输出地址 -t 类型
python ToLocalizable.py -f xlsx地址 -s 输出地址 -t 类型
